============================================================ STIMA SANTUY ============================================================

How to Run:
1) Go to your command prompt and set path to where Game Engine in Game Engine version xx appear
   In example : cd C:\Users\manasyebukit\Documents\GitHub\battleship-greedy-santuy\src\Game Engine -  Version 1.2.2\Game Engine
2) To play type this command : Battleships.exe -b "..\Reference Bots\Python3" "..\..\Our Bot" --clog --pretty 
   First string "..\Reference Bots\Python3" for bot player 1
   Second sting "..\..\Our Bot" for bot player 2
3) To change map size : Battleships.exe -s <idx> -b "..\Reference Bots\Python3" "..\..\Our Bot" --clog --pretty 
   idx = 1 -> small map with map size 7x7
   idx = 2 -> medium map with map size 10x10
   idx = 3 -> large map with map size 14x14
4) Now you can play and enjoy the most astonishing battleship ever
